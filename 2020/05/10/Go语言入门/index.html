Go语言入门

发表于 2020-05-10 | 分类于 Go |

变量

var name type = expression

Go 语言的类型在变量后面。
类型和表达式部分可以省略一个。

  1. 如果类型省略，类型由初始值表达式决定；

  2. 如果表达式省略，其初始值为类型的零值。

    数字 = 0
    布尔值 = false
    字符串 = “”
    接口和引用类型（slice、指针、map、通道、函数）= nil
    想数组或结构体这样的复合类型 = 所有元素或成员的零值

短变量声明

短变量：可以用来声明和初始化局部变量。
name := expression

短变量声明：短小、灵活，在局部变量的声明和初始化中主要使用短声明。
var声明：通常是为那些跟初始化表达式类型不一致的局部变量保留的，或者用于后面才对变量赋值以及变量初始化不重要的情况。

段变量声明不需要声明所有在左边的变量，如果一些变量已经在前面声明，那么对于这些变量，短声明的行为相当于赋值。但是短变量声明最少需要声明一个新变量，否则编译不通过。

指针

指针的值是一个变量的地址（变量保存的地址）。
不是所有的值都有地址，但是所有的变量都有。
使用指针，可以在无须知道变量名字的情况下，间接读取或更新变量的值。

new函数

表达式new(T)创建一个未命名的T类型变量，初始化为T类型的零值，并返回其地址（地址类型为*T）。

每一次调用new返回一个具有唯一地址的不同变量。但是两个变量的类型不携带任何信息且是零值，例如struct{}或[0]int，当前的实现里面，他们有相同的地址。

类型声明

type name underlying-type
类型的声明通常出现在包级别，这里命名的类型在整个包中可见。如果名字开头大写，其他的包也可以访问它。

如果类型/接口/方法/函数/字段的首字母大写，则是 Public 的，对其他 package 可见;
如果首字母小写，则是 Private 的，对其他 package 不可见。

类型声明提供了概念上的便利，避免了一遍遍地重复写复杂的类型，适用于复杂结构体类型。

错误处理

go中defer,panic,recover三者一起使用可以充当类似java中try…catch…的角色，而defer本身又像其他语言的析构函数。
defer关键字可以让函数或语句延迟到函数语句块的最结尾时执行。
含有defer表达式的函数返回的过程：先给返回值赋值，然后调用defer表达式，最后才是返回到调用函数中。

defer函数的参数值，是在申明defer时确定下来的
在defer函数申明时，对外部变量的引用是有两种方式：作为函数参数和作为闭包引用
作为函数参数，在defer申明时就把值传递给defer，并将值缓存起来，调用defer的时候使用缓存的值进行计算。
作为闭包引用，在defer函数执行时根据整个上下文确定当前的值。

并发编程

goroutine

Go中，每个并发执行的活动称为goroutine。
go f(),go语句使函数在一个新创建的goroutine中调用。

通道

通道是goroutine之间的连接。
通道两个主要的操作：发送和接收，两者统称为通信。
可以在goroutine之间传递消息。阻塞等待并发协程返回消息。

sync

Go 提供了 sync 和 channel 两种方式支持协程(goroutine)的并发。
